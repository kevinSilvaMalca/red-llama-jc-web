# the-triple-three-website

WebSite to The Triple Three

## Configure project

```
Node version: 20.16.0
Vite version: 5.3.4
React version: 18
```

## Installations

### Install Node.js

```
nvm install 20.16.0
```

### Create app in repo folder

```
npm create vite@latest . --template react
```

### Install dependencies

```
npm install react-router-dom
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
npm install prop-types
npm install react-calendar
npm install axios
```

[Install Tailwind CSS doc](https://tailwindcss.com/docs/guides/vite)

## Run project

```
npm install
npm run dev
```
